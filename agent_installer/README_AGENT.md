# Voltera Agent — Installatie

1. Dubbelklik `install.py`
2. Vul het serveradres in (ontvang je van de beheerder, bijv. `https://abc123.ngrok-free.app`)
3. Klik op **Installeren** — klaar
4. De agent draait nu op de achtergrond en start automatisch bij elke Windows-login
5. Bij problemen: stuur het bestand `%LOCALAPPDATA%\VolteraAgent\agent.log` naar de beheerder
